# 什么是前端?

前端（Front-end） 指的是互联网产品中直接面向用户的部分。它是用户能看到、听到、触摸到并进行交互的界面总和，也被称为“客户端”。

## 前端的应用领域

- Web
- 移动端 App
- 桌面端 App
- 小程序
- CLI
- 游戏
